Comandos Basicos de Git:
git status:
Muestra el resumen en forma de lista del estado, en el que se encuentran los archivos que tienen cambios.

git add: 
Agrega los archivos actuales en los que se esta trabajando ya sea nuevo o modificado.

git commit:
se usa para agregar una descripción o comentario de los archivos que deseaas agregar al repositorio.

git pull:
Se usa para juntar o fusionar los cambios que se hacen en el repositorio.

git log:
Muestra todos los commits en forma de lista, seguido de una descripcion de cada commit.

git remote:
se usa para conetarse a un repositorio remoto.

git gc:
optimiza el repositorio por medio de recolección de basura, limpia los archivos innecesarios.
