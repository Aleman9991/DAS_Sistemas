Git Status:
Muestra los cambios del repositorio actual, si está al día con tu repositorio remoto, notifica de cambios por agregar,
commits pendientes de hacer o elementos arvhivos no trackeados

Git add:
Añade archivo/s para poder "trackearlos" para poder hacer commit después

Git commit:
Guarda los archivos agregados en tu repositorio local, se puede decir que son los cambios, y pueden ser de uno o mas archivos
dependiendo cuantos se agregaron antes.

Git push:
Sube los cambios del repositorio local al remoto destino, indicando tambien la rama a la cual quieres subir cambios.

Git log:
Lista de los commits del repositorio mostrando fecha, hora, mensaje y un id.

git remote add:
Agrega un repositorio remoto, generalmente se agrega un repositorio "upstream" para hacer updates.

git pull:
Hace fetch y merge desde un repositorio a tu repositorio local, indicando en el comando el nombre del remoto y la rama
a actualizar.
