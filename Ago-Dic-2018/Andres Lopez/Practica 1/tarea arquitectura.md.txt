git status:
te dice que archivos han sido modificados y que archivos se quedaron sin modificar

git add:
sirve para añadir archivos a tu carpeta de trabajo, para despues hacer un commit y guardar las modificaciones

git commit:
guarda los cambios de los archivos para despues subirlos a Github con un push

git pull:
trae los cambios de un repositorio a tu computadora

git push:
es para subir el archivo a Github

git log:
te enlista los commit en el repositorio
